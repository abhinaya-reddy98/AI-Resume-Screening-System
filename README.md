# AI Resume Screening System

An AI-powered Resume Screening System built using Python, Streamlit, and Google Gemini AI.

## Features

* Resume PDF Upload
* Resume Text Extraction
* ATS Score Calculation
* Skill Matching
* Missing Skill Detection
* Resume Improvement Suggestions
* Career Guidance Engine
* Gemini AI Resume Analysis
* Downloadable Resume Report

## Tech Stack

* Python
* Streamlit
* Google Gemini AI
* PyPDF

## How It Works

1. Upload a resume PDF.
2. Paste a job description.
3. The system extracts skills from the resume.
4. ATS score is calculated based on job requirements.
5. Missing skills and improvement suggestions are displayed.
6. Gemini AI provides detailed feedback and recommendations.
7. Download a complete analysis report.
